# Built & Company Website

A Modern, Responsive landing page for Built & Company.

## About

This is a frontend-only React application showcasing Built & Company's services and portfolio. The site features smooth animations, a clean design, and works well on both mobile and desktop.

## Tech Stack

- React
- Tailwind CSS
- AOS (Animate On Scroll)
- React Multi Carousel

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## What's Inside

The site includes sections for:
- Hero banner with company branding
- Services overview (Lighting, Rigging, A/V, Transportation, etc.)
- Project carousel
- Stats and achievements
- Contact information

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.
